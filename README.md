# Agenda Personal 

Una aplicación de escritorio sencilla e intuitiva desarrollada en Python para gestionar eventos personales. Permite agregar y eliminar recordatorios con fecha, hora y descripción mediante una interfaz gráfica.

## Características 
* **Añadir eventos:** Selecciona la fecha fácilmente con un calendario interactivo (`DatePicker`).
* **Visualización clara:** Lista organizada de todos tus eventos programados en formato de tabla.
* **Eliminación segura:** Borra fácilmente las tareas con un mensaje de confirmación para evitar accidentes.
* **Interfaz amigable:** Construida de forma nativa utilizando `tkinter`.

## Requisitos 
Para que el programa funcione correctamente en tu entorno local, necesitas tener instalado:
* Python 3.x
* Librería `tkcalendar`
